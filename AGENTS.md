## NEVER use destructive git commands
- JANGAN PERNAH gunakan `git checkout -- .`, `git reset --hard`, `git restore .`, atau perintah git destruktif lain yang bisa menghapus perubahan yang belum dicommit.
- Semua perubahan kode sangat berharga dan TIDAK BOLEH di-reset dalam keadaan apapun.
- Jika perlu revert perubahan, lakukan secara manual dengan mengedit file, bukan dengan git checkout/reset.
